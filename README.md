## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` folder.

### Preview Production Build

```bash
npm run preview
```

## Customization

### Update Personal Information

Edit the following files to update your information:
- `src/pages/Home.jsx` - Name, role, and intro
- `src/pages/About.jsx` - About section content
- `src/pages/Contact.jsx` - Contact information
- `src/components/Footer.jsx` - Social links and address

### Update Projects

Edit `src/pages/Projects.jsx` to add or modify projects.

### Update Skills

Edit `src/pages/Skills.jsx` to modify skills and proficiency levels.

### Update Experience

Edit `src/pages/Experience.jsx` to update work experience and education.

### Change Color Theme

Edit `tailwind.config.js` to customize the color scheme.

## Resume Download

The resume PDF is located in the `public` folder and can be downloaded via the "Download Resume" button in the header and home section.
